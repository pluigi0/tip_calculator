# Tip Calculator

A clean, minimal tip calculator built with vanilla HTML, CSS, and JavaScript. No frameworks, no dependencies — just a single file ready to use.

## Live Demo

[View it live](https://github.com/pluigi0/tip_calculator) 

## Features

- Preset tip buttons: 10%, 15%, 20%, 25%
- Custom tip percentage input
- Bill split across multiple people
- Real-time calculation as you type
- Reset button to clear all fields

## Tech Stack

- HTML5
- CSS3 (custom properties, Flexbox, Grid)
- Vanilla JavaScript (DOM manipulation, event listeners)

## Getting Started

No installation needed. Just open `index.html` in your browser.

```bash
git clone https://github.com/pluigi0/tip_calculator
cd tip-calculator
open index.html
```
